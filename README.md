# Zaawansowane Aplikacje Webowe

## Opis projektu

Celem projektu jest praktyczna realizacja efektów kształcenia określonych w podstawie programowej kwalifikacji **INF.04 – Projektowanie, programowanie i testowanie aplikacji**, ze szczególnym uwzględnieniem tworzenia nowoczesnych aplikacji internetowych wykorzystujących technologie frontendowe, backendowe oraz bazy danych.

## Cele projektu

W ramach projektu uczniowie:

- analizują wymagania funkcjonalne i niefunkcjonalne aplikacji,
- projektują interfejs użytkownika zgodnie z zasadami UX/UI,
- tworzą aplikacje internetowe z wykorzystaniem współczesnych technologii webowych,
- implementują komunikację klient–serwer,
- projektują i obsługują relacyjne lub nierelacyjne bazy danych,
- stosują wzorce projektowe i dobre praktyki programistyczne,
- wykonują testy aplikacji,
- dokumentują proces tworzenia oprogramowania,
- korzystają z systemów kontroli wersji.

## Zakres realizowanych zagadnień

### Frontend

- HTML5
- CSS3
- JavaScript (ES6+)
- TypeScript
- Frameworky frontendowe (Angular, React lub Vue)
- Responsywny interfejs użytkownika (RWD)

### Backend

- ASP.NET Core
- Node.js
- REST API
- Autoryzacja i uwierzytelnianie użytkowników
- Obsługa plików i danych

### Bazy danych

- MySQL
- PostgreSQL
- Microsoft SQL Server
- MongoDB
- ORM (Entity Framework Core)

### Narzędzia

- Git
- GitHub
- Docker
- Visual Studio / Visual Studio Code
- Postman

## Efekty kształcenia INF.04 realizowane w projekcie

### Projektowanie aplikacji

- analiza wymagań użytkownika,
- tworzenie dokumentacji projektowej,
- projektowanie architektury systemu.

### Programowanie aplikacji

- implementacja logiki biznesowej,
- tworzenie komponentów interfejsu użytkownika,
- integracja z bazami danych,
- tworzenie usług sieciowych.

### Testowanie aplikacji

- przygotowywanie scenariuszy testowych,
- wykonywanie testów funkcjonalnych,
- identyfikacja i usuwanie błędów,
- walidacja poprawności działania aplikacji.

### Zarządzanie projektem

- planowanie etapów realizacji,
- praca zespołowa z wykorzystaniem Git,
- dokumentowanie postępów prac.
